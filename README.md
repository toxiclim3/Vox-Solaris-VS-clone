## This game is the work of team Vox Solaris <sub>(Warframe reference <3)</sub>   
Made as part of a groupwork in Uni <sub>that apparently I'm doing alone</sub>  
This is a Vampire Survivors inspired game, ~~still without a name~~ now called Inritus!  
Made in Godot 4.6  

At [Audio/Music](Audio/Music), there are empty folders for you to add your own music (since the ones i'm using are probably copyrighted)
Boss and Normal automatically use what's in the folder
Extra needs manual assignment in [Utility/music_controller.gd](Utility/music_controller.gd)
